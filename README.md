# coffee-shop-teamwork
项目简介
本项目是 CSC8019 课程小组作业，开发了一个咖啡店在线点单系统，支持用户浏览菜单、在线下单、查看及管理订单状态。



技术栈

- 前端：HTML / CSS / JavaScript

- 后端：Node.js + Express

- 数据库：MySQL（云端部署）

- 版本控制：GitHub



云端服务配置

- 服务规格：1 CPU / 1 GB RAM / 1 GB 存储

- 资源使用：内存 71%，存储 28.8%，CPU 占用 < 5%



快速开始

1. 克隆项目

git clone https://github.com/chaowang55/coffee-shop-teamwork.git

2. 进入项目目录

cd coffee-shop-teamwork

3. 打开Vscode安装依赖，一下所有的操作都在终端中完成，终端可以通过听不得更多打开终端： 终端----新的终端

npm install

4. 启动后端服务

node server.js



组员协作

- 所有代码托管在 GitHub，组员通过 git pull 获取最新版本。

- 数据库已部署到云端，组员无需本地安装 MySQL，直接运行即可连接。

Project Overview

This project is a group assignment for CSC8019, which develops an online coffee shop ordering system. It supports users to browse the menu, place orders online, and view or manage order statuses.



Tech Stack

- Frontend: HTML / CSS / JavaScript

- Backend: Node.js + Express

- Database: MySQL (cloud deployment)

- Version Control: GitHub



Cloud Service Configuration

- Service Plan: 1 CPU / 1 GB RAM / 1 GB storage

- Resource Usage: Memory 71%, Storage 28.8%, CPU < 5%



Quick Start

1. Clone the repository

git clone https://github.com/chaowang55/coffee-shop-teamwork.git

2. Navigate to the project directory

cd coffee-shop-teamwork

3. Open Vscode to install dependencies. All the following operations are completed in the terminal. The terminal can be opened through Listen No More: Terminal ---- New Terminal

npm install

4. Start the backend server

node server.js





Team Collaboration

- All code is hosted on GitHub. Team members can get the latest version via git pull.

- The database is deployed on the cloud. No local MySQL installation is required for team members; run the server directly to connect.
